# Commercial Licensing

`sharepoint-rag-lite` is distributed under a non-commercial license by default.
If you want to use this project for commercial purposes, you need a separate
commercial agreement.

## Examples that require commercial licensing

- Operating this project as a paid SaaS or managed service
- Including this project in a paid product, package, or deliverable
- Internal production use directly tied to revenue-generating operations
- Redistributing modified versions as part of commercial offerings

## What to include in your request

- Organization and contact details
- Intended use (SaaS, on-prem package, consulting, internal platform, etc.)
- Scale (users, tenants, expected traffic)
- Whether source code modification and redistribution are planned
