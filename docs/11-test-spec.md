## シート: はじめに

### 概要

本書は、SharePoint RAG Lite PoC の試験仕様を記載した資料である。既存構成（AI Search 版）の試験仕様 18 ケースを全て引き継ぎ、Lite 構成固有の追加ケースを含む。

### 前提条件

- Supabase セットアップ完了（pgvector 有効化 + テーブル作成）
- Azure リソース構築完了（10-build-guide.md Step 1〜2）
- インジェスション完了（10-build-guide.md Step 4）
- テストユーザー 3名のアカウントが利用可能
