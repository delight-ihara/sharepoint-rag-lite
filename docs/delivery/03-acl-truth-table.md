# ACL 真理値表 — テンプレート

> 「誰が何を見れるか」を構築前に顧客と合意する書面。
> この表がテストの合否判定基準となる。合意なしに構築を開始しない。

## 記入ルール

1. フォルダ名を左列に列挙
2. ユーザー/グループを上部に列挙
3. ○ = 閲覧可、× = 閲覧不可 を記入
4. 記入後、顧客担当者の確認を得る（メール or 署名）
